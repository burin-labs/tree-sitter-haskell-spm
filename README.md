# tree-sitter-haskell-spm

Swift Package Manager wrapper for the [tree-sitter-haskell](https://github.com/tree-sitter/tree-sitter-haskell) grammar.

## Usage

Add this package to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/burin-labs/tree-sitter-haskell-spm.git", from: "0.1.0")
]
```

Then add `TreeSitterHaskell` as a dependency of your target.
